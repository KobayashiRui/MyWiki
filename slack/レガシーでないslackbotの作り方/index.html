ディレクトリ構成

slackbot  ⇒ プログラムをまとめるディレクトリ名前は任意

  • run.py ⇒ このプログラムを実行することでボットを起動する
  • slackbot_settings.py ⇒ botに関する設定を書くファイル
  • plugins ⇒ botの機能はこのディレクトリに追加する
  • __init__.py ⇒ モジュールを示すためのファイル 空でよい
  • my_mention.py ⇒ 機能を保存する 任意の名前でOK

run.pyの例

from slackbot.bot import Bot

def main():

        bot = Bot()

        bot.run()

if __name__ == “__main__”:

        print(“start slackbot”)

        main()

slackbot_settings.py

API_TOKEN = “botアカウントのトークン”

DEFAULT_REPLY = “” #bot宛のメッセージでどの応答にも当てはまらない場合の応答文字列

PLUGINS = [‘plugins’] #プラグインスクリプトを置いてあるサブディレクトリ名のリスト

@default_reply()

def default_func(message):

        text = message.body['text']         # メッセージを取り出す

        # 送信メッセージを作る。改行やトリプルバッククォートで囲む表現も可能

        print("data")

        print(message.body)

        if 'files' in message.body:

            url = message.body['files'][0]['url_private']

            flag = message.body['files'][0]['filetype']

            tmpfile = "./tmp." + flag

            token = 'xoxb-359991097699-553217769011-l5OVyxAoGiRndwTyzz4I0hjs'

            rst = requests.get(url, headers={'Authorization': 'Bearer %s' % token}, stream=True)

            fo = open(tmpfile, "wb")

            shutil.copyfileobj(rst.raw, fo)

            fo.close()
